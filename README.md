# Разработаны классы Product и Category с необходимыми атрибутами и методами 

## Реализована автоматическая обработка атрибутов класса для учета общего количества категорий и товаров.

## Добавлена поддержка загрузки данных из файла формата JSON.

## Проведено покрытие функциональности тестами для подтверждения работоспособности.

## Создание абстрактного класса: Введён базовый абстрактный класс BaseProduct, служащий фундаментом для всех типов продуктов. 

## Использование миксинов: Реализован класс-миксин LogMixin, обеспечивающий удобный механизм регистрации событий при создании объектов.

## Разделение ответственности: Для каждой группы товаров созданы собственные подклассы, наследующие базовый класс и миксин.
